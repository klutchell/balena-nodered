# resin-nodered

[resin.io](https://resin.io/) stack with the following services:
* [node-red iot interface](https://nodered.org/)
* [cloud9 web ide](c9.io)
* [ssh server](https://www.ssh.com/ssh/)

## Getting Started

* https://docs.resin.io/learn/getting-started
* https://nodered.org/docs/getting-started

## Deployment

```yaml
# example docker-compose.yml
version: '2.1'

volumes: 
  node-red-data:
  ssh-data:
  
services:

  node-red:
    build: ./node-red
    ports:
      - '80:1880'
    volumes: 
      - 'node-red-data:/data'

  ssh:
    build: ./ssh
    ports:
      - '22:22'
    volumes: 
      - 'node-red-data:/data'
      - 'ssh-data:/root/.ssh'

  cloud9:
    build: ./cloud9
    ports:
      - '8080:8080'
    volumes: 
      - 'node-red-data:/data'
      - 'ssh-data:/root/.ssh'
```
